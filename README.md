# Sidro Carousel


Carousel and Slide with thumbmail Vue component


## Install

```bash
yarn add @sidro/carousel
```

```bash
npm install @sidro/carousel
```

## Import &  Usage

```js
import '@sidro/carousel'
```

```js
<s-carousel>
      <s-slide id="0" src="IMAGE-URL" ></s-slide>
      <s-slide id="1" src="IMAGE-URL" ></s-slide>
</s-carousel>
```
